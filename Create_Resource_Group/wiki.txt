Creates a resource group in Azure.



Step 1:
Authenticate to Azure using Azure CLI. Terraform only supports Azure CLI. Authentication via powershell is not supported.

Run "az login" to Authenticate
Run "az account show" to confirm the current azure subscription
Run "az account list" to list all the subscriptions
Run "az account set --subscription <Subscription Name or Subcription ID>" to use a specific Azure subscription
Run "az account show" command to confirm subscription.
